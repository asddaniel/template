    Developpement Web avec Python et Django
    Django est un framework Web Python de haut niveau qui encourage un développement rapide et une conception propre et pragmatique. Construit par des développeurs expérimentés, il prend en charge une grande partie des tracas du développement Web, vous pouvez donc vous concentrer sur l'écriture de votre application sans avoir à réinventer la roue. C'est gratuit et open source.
    Developpement Web avec Python et Flask
    Flask est considéré comme plus Pythonic que le framework Web Django car dans des situations courantes, l'application Web Flask équivalente est plus explicite. Flask est également facile à utiliser en tant que débutant, car il existe peu de code passe-partout pour obtenir une application simple et opérationnelle.
